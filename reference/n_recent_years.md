# Get the most recent years of data Assumes the df being filtered was created by \`save_history()\`

Get the most recent years of data Assumes the df being filtered was
created by \`save_history()\`

## Usage

``` r
n_recent_years(history, n = 5)
```

## Arguments

- history:

  A data frame containing marks history, as created by
  \[norman::save_history()\].

- n:

  integer - the number of most recent years
