# Get the module effect for a given module code

Get the module effect for a given module code

## Usage

``` r
print_module_effect(module_code, mdf)
```

## Arguments

- module_code:

  Character; the length-5 module code

- mdf:

  Object of class `lm`, the module fitted to median differences

## Value

Character; the module effect, rounded to 1 decimal place.
