# Check the marks folder contents against file "modules_expected_here.txt"

Check the marks folder contents against file "modules_expected_here.txt"

## Usage

``` r
check_modules_expected(working_directory, module_codes)
```

## Arguments

- working_directory:

  Character; the full path to the folder for this report

- module_codes:

  Character; the vector of module codes used in the report

## Value

A list with two character components, named `extras` and `missing`
