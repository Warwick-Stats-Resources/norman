# Make a matrix of percentages in each degree class range, for all modules

Make a matrix of percentages in each degree class range, for all modules

## Usage

``` r
raw_mark_classes(marks_matrix, dp = 0)
```

## Arguments

- marks_matrix:

  The full matrix of marks for the modules

- dp:

  Numeric, the number of decimal places to round the result to

## Value

A matrix
